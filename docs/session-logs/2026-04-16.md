## Key insights
- The previous "drift report" from 2026-04-15 was a login-page comparison.
  That entire analysis needs to be mentally discarded — only today's report
  represents real Figma-vs-DOM drift.
- Modern SPAs rarely use cookie-only auth. Cookie-header approach was doomed
  for this stack; storageState is the right default going forward.
- Color match rate of 41% exact on first authenticated run is actually a
  strong signal — the brand ships correctly where it ships at all.
- Typography 100% within-family (all Nunito) confirms the type system has
  no family drift — any typography drift is size/weight, which is fixable
  mechanically by capping size/weight permutations in CSS.
